# MeshLib
A kotlin library for dealing with 3d meshes.

## How to get
Gradle (kts):
```kotlin
repositories {
    maven {
        name = "alex's repo"
        url = uri("http://207.180.202.42:8080/libs")
        isAllowInsecureProtocol = true
    }
}

dependencies {
    implementation("me.alex_s168:meshlib:0.8")
    implementation("me.alex_s168:mathlib:0.5h6")
}
```
